TypeScript

Présentation

TypeScript est un langage de programmation open-source développé par Microsoft. Il est un sur-ensemble de JavaScript et ajoute des fonctionnalités de typage statique au langage.

TypeScript est conçu pour améliorer la productivité des développeurs en fournissant une vérification de type au moment de la compilation, ce qui permet de détecter les erreurs avant l'exécution du code.

Popularité

D'après le Stack Overflow Developer Survey 2024, TypeScript est classé à la cinquième place des langages de programmation les plus populaires derrière JavaScript, HTML/CSS, Python et SQL.

Stack Overflow Developer Survey 2024

Compilation

TypeScript est un langage compilé, ce qui signifie que le code TypeScript doit être transformé en JavaScript avant d'être exécuté dans un environnement JavaScript (on parle également de transpilation).

Pour compiler le code TypeScript, il est possible d'installer TypeScript via npm : 

npm install typescript

Il est alors possible de compiler un fichier TypeScript en JavaScript en utilisant la commande suivante : 

npx tsc fichier.ts

Cependant, nous allons utiliser une méthode plus simple pour créer notre projet TypeScript en utilisant Vite : 

npm create vite@latest

Ensuite, il suffit de choisir le framework Vanilla et de sélectionner TypeScript comme langage.

Vite type checking

Par défaut, Vite ne vérifie pas le typage de TypeScript lors du développement et va uniquement effectuer la transpilation du code TypeScript en JavaScript (il va laisser l'IDE se charger du typage).

Pour activer la vérification de type, il est possible d'utiliser le plugin vite-plugin-checker : 

npm install vite-plugin-checker --save-dev

Ensuite, il suffit de l'ajouter dans le fichier de configuration de Vite vite.config.ts : 

import checker from 'vite-plugin-checker';
export default {
  plugins: [
    checker({ typescript: true }),
  ],
}

Utiliser les fonctionnalités avancées de JavaScript

Le langage TypeScript étant compilé en JavaScript, il permet d'utiliser les dernières fonctionnalités de JavaScript, même si elles ne sont pas encore prises en charge par tous les navigateurs.

Il est même possible de définir la version de JavaScript cible dans le fichier de configuration tsconfig.json en utilisant la propriété target. 

{
    "compilerOptions": {
        "target": "es2020"
    }
}
TypeScript Configuration Target

Définir les librairies utilisées

Il est également possible de définir les librairies utilisées dans le projet en utilisant la propriété lib dans le fichier de configuration tsconfig.json.

Par exemple, pour utiliser les fonctionnalités de la librairie DOM, il suffit d'ajouter "dom" dans la propriété lib : 

{
    "compilerOptions": {
        "lib": ["es2020", "dom"]
    }
}

La librairie DOM est utilisée pour manipuler le Document Object Model (DOM) dans le navigateur et est indispensable pour le développement d'applications web.

Sans cette librairie, TypeScript ne reconnaîtra pas les objets du DOM tels que document ou window.
